# Data visualizator

Application for the visualization of simple graphs

## Build

Build the project using the scripts inside the `scripts/`:

```sh
# Build the project
./scripts/build.sh

# Remove all build files
./scripts/clear.sh
```

## Requirements

- cmake 3.22.1
- make 4.3
- gcc 11.4.0
- c++ std=17
