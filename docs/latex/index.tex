 Small application for the visualization of data generating simple graphs. All of this with the objective of learning and applying multiple C++ concepts.\hypertarget{index_autotoc_md1}{}\doxysection{Build}\label{index_autotoc_md1}
Build the project using the scripts inside the {\ttfamily scripts/} directory (execute them on the main project\textquotesingle{}s directory)\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\# Build the project (generate executables and documentation)}
\DoxyCodeLine{./scripts/build.sh}
\DoxyCodeLine{}
\DoxyCodeLine{\# Create make files}
\DoxyCodeLine{./scripts/make.sh}
\DoxyCodeLine{}
\DoxyCodeLine{\# Generate all executables (main program and tests)}
\DoxyCodeLine{./scripts/cmake.sh}
\DoxyCodeLine{}
\DoxyCodeLine{\# Generate main executable}
\DoxyCodeLine{./scripts/cmake.sh visualizator.exe}
\DoxyCodeLine{}
\DoxyCodeLine{\# Generate tests executables}
\DoxyCodeLine{./scripts/cmake.sh tests.exe}
\DoxyCodeLine{}
\DoxyCodeLine{\# Remove all build files (executables, build/ files and doxygen documentation)}
\DoxyCodeLine{./scripts/clear.sh}

\end{DoxyCode}
\hypertarget{index_autotoc_md2}{}\doxysection{Execution}\label{index_autotoc_md2}
Execute the program using the following command\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\# Execution of program}
\DoxyCodeLine{./visualizator.exe [relativePathToFile/] }
\DoxyCodeLine{}
\DoxyCodeLine{\# Example of execution with example file}
\DoxyCodeLine{./visualizator.exe data/test/test-\/dataset.csv }

\end{DoxyCode}
